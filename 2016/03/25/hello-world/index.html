Hexo你的博客

发表于 2016-03-25   |   分类于 hexo   |     |   阅读次数  

欢迎来到 Hexo! 这里我们将介绍如何搭建自己酷炫的博客（by hexo）。
访问 documentation 以获取更多信息. 使用hexo途中遇到的问题都可以访问 troubleshooting 或咨询 GitHub.

快速开始

本地安装node，完成后全局安装hexo

创建第一篇文章

在根目录下的source/_posts下找到它，打开后编辑，编写是用markdown语法编写的哦，不熟悉的话可以先上百度瞧一瞧。

生成静态页面

根据编写好的My-New-Post.md，快速生成静态的html

开启本地服务器

接下来打开浏览器，输入127.0.0.1:4000。就可以看到新创建好了的文章了~
怎么样，是不是很震撼~，接下来我们准备把它放到远程服务器上，这样就可以在线观看了。

申请github账号

前往官网申请github账号，然后注册一个域名（如：myName.github.io），同步设置本地SSH key和github上的SSH key

把文章hexo到github上面去，

大功告成~

虽说是大功告成了，但我们的blog还需要再酷炫一点呀

更换hexo主题

先上网git下来一个主题（lz在这里推荐很好看的next，pacman主题），放在根目录的themes文件夹下，修改根目录下的_config.yml配置文件

在文章中插入图片

推荐使用七牛云，登录七牛云官网（使用旧版本），将图片上传至服务器，点击图片的下拉按钮以拷贝外链地址，
在文章中使用markdown引入图片的语法

添加评论功能

注册多说账号，生成通用代码块，拷贝进next相应位置，重新生成页面

补充分类，标签页为空的问题

本文完，有疑问可尽情留言~
